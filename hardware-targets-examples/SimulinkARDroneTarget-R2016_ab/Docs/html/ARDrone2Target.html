AR Drone 2.0 Target

The AR Drone 2.0 Target provides hardware support for the Parrot AR Drone 2.0, a commercially available quad-copter.

Contents:

Concepts

Learn about how the AR Drone 2.0 functions, driver support is achieved and code generation works for the AR Drone 2.0 Target.

Demo

The demo for the 2016 MATLAB Benelux EXPO is included with the AR Drone 2.0 Target. Provided you have the needed materials you can follow the step by step instructions to run the demo.

Examples

The examples section explains how to get models running on the AR Drone 2.0. The library blocks provided with the AR Drone 2.0 Target are used in the examples to show you how they can be used for things such as streaming video or hovering.

Documentation

This section contains links to external information resources about the drone hardware and software, release notes and an FAQ/troubleshooting section.
