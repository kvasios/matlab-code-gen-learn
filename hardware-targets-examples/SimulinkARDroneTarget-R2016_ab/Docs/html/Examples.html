AR Drone 2.0 Target: Examples

The provided examples will guide you through the capabilities of the AR Drone 2.0 Target. The examples are listed in increasing difficulty, ranging from checking if your drone runs the supported firmware up to free flight of the drone.

Getting started

This example shows how to get a simple Simulink model running on the drone in external mode. The model will verify whether the firmware on your drone is supported.

Configuring a model for the AR Drone 2.0

The process of running your own models on the AR Drone 2.0 is explained step by step in this example.

Streaming video in external mode

This example will show you how to show the video feed from the AR Drone 2.0 within Simulink using External mode.

Streaming video in Normal mode

This example will show you how to run a model in normal and stream video data to the PC using UDP.

Reading and logging sensor data

In this example the data from the various sensors on the AR Drone 2.0 will be read into a program created from a Simulink model, allowing visualisation in real time and logging to the Simulink PC.

Calibrating the sensors

This example will show you how to use the GUI provided to calibrate the Intertial Measurement Unit sensors.

Color detection

The color detection example shows how to calibrate parameters such that you can determine where in the camera view, an object of a certain color lies.
